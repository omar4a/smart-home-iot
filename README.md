# Embedded Systems Project

This repository contains source files for an embedded systems project, including C code and supporting files generated by IAR Embedded Workbench.

## Overview

- `main.c`, `project.c`, and other source files implement the microcontroller application logic.
- The `IAR/` and `simulator-iar/` directories include workspace and build files for IAR simulator setup.

## Documentation

Project details are included in the PDF named `Embedded_Report`.

## Notes

- Configure your toolchain (IAR, SDK, etc.) before building.
- Build and run from the IAR project in `IAR/simulator-iar`.
