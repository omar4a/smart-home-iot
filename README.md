# Smart Home IoT Prototype 🏠🔌

Smart Home IoT prototype on ARM Cortex-M4 (TM4C123GH6PM).

## 🌟 Features
- Mobile-controlled lighting and sensors
- Fire alarm system
- Temperature monitoring with SPI/I2C communication
- Modular peripheral integration

## 🛠️ Technologies
`C` `ARM Cortex-M4` `TM4C123GH6PM` `SPI` `I2C` `UART` `IoT`
